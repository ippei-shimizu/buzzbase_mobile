## App Store 説明文 (iOS, 4000文字以内)

BUZZ BASEは、野球の個人成績を記録・管理し、仲間とランキング形式で競い合える野球成績アプリです。

打率や防御率の計算はアプリにおまかせ。試合結果を入力するだけで、29種類の成績指標を自動で算出します。さらに、グループ機能でチームメイトやライバルと成績をランキングで比較できる、他にはない機能を搭載しています。

中学・高校・大学の野球部員から草野球プレーヤーまで、すべての野球人のための成績管理アプリです。

主な機能
━━━━━━━━━━━━━━━━

【試合結果の記録】
試合ごとに打撃成績・投手成績を記録できます。打席結果をタップするだけのカンタン入力。対戦相手、スコア、打順、守備位置も記録できます。

【29種類の成績を自動計算】
打率・出塁率・長打率・OPS・ISO・ISOD・BB/K・盗塁成功率など打撃14指標、防御率・WHIP・K/9・BB/9・K/BB・勝率など投手15指標を自動で算出。面倒な計算は一切不要です。

【ダッシュボード】
自分の打撃成績・投手成績のサマリーをひと目で確認。レーダーチャートで自分の強み・弱みを視覚的に把握できます。年度・シーズン・試合種別ごとにフィルタリングも可能。

【グループランキング機能】
チームメイトや野球仲間でグループを作成し、打率・打点・本塁打・防御率・奪三振などの成績をランキング形式で比較できます。同じチーム内はもちろん、他チームのライバルとも競い合えます。

【シーズン管理】
年度やシーズンごとに成績を分けて管理。公式戦・オープン戦の区別も可能。シーズンを振り返りながら、自分の成長を実感できます。

【試合結果一覧】
すべての試合記録を一覧で確認。対戦相手で検索したり、日付・シーズン・試合種別でフィルタリングできます。

【プロフィール】
ポジション・チーム情報・受賞歴を登録して、自分の野球プロフィールを作成。成績とあわせて他のユーザーに公開できます。

【フォロー機能】
気になる選手やチームメイトをフォロー。フォローしたユーザーの成績をチェックできます。

【野球ノート】
試合の振り返りや練習メモを記録。成績データとあわせて、自分の課題や気づきを書き留められます。

【通知機能】
フォローやグループ招待の通知をリアルタイムで受け取れます。

【プライバシー設定】
非公開アカウントに設定すれば、承認したフォロワーだけが成績を閲覧できます。

### BUZZ BASEの差別化ポイント

1. **ランキング機能**: 競合5社すべてにグループ内成績ランキング機能がない。唯一の差別化ポイント
2. **チーム横断グループ**: 他チームのライバルとも比較できる設計は競合ゼロ
3. **投打29種の成績計算**: 打撃特化のヒットメーカーより投手成績の網羅性で優位
4. **完全無料**: 競合は課金あり（スコアラー1,800円、PLAY月額課金等）

### 競合の共通訴求パターン

- 「無料」「簡単入力」「自動計算」「セイバーメトリクス対応」が定番
- 全競合が「記録・管理・分析」の機能訴求に終始
- BUZZ BASEは「仲間と競い合う・モチベーションが上がる」の体験価値で差別化
